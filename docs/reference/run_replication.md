# Run a named replication or prep step

Run a named replication or prep step

## Usage

``` r
run_replication(id, install_deps = FALSE)
```

## Arguments

- id:

  Replication id from `replication.yml` (e.g. `"fig_1"`, `"prep_data"`).

- install_deps:

  Ignored; kept for compatibility with replicateEverything.

## Value

Analysis or display object.
