# Figure 1 — distribution of support for contributions of different sizes

Figure 1 — distribution of support for contributions of different sizes

## Usage

``` r
make_figure_1(data = wave4_conjoint)

make_fig_1(data = wave4_conjoint)
```

## Arguments

- data:

  Wave 4 conjoint data. Defaults to
  [wave4_conjoint](https://replicate-anything.github.io/rep-10.1371-journal.pone.0278337/reference/wave4_conjoint.md).

## Value

A multi-panel ggplot object from
[`egg::ggarrange`](https://rdrr.io/pkg/egg/man/ggarrange.html).
