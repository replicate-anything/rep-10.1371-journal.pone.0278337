# Table 1 — structural parameter estimates

Table 1 — structural parameter estimates

## Usage

``` r
make_table_1(data = wave4_conjoint)

make_tab_1(data = wave4_conjoint)
```

## Arguments

- data:

  Wave 4 conjoint data. Defaults to
  [wave4_conjoint](https://replicate-anything.github.io/rep-10.1371-journal.pone.0278337/reference/wave4_conjoint.md).

## Value

A data frame of parameter estimates with confidence intervals.
