# rep1371journalpone0278337: Replication Package for Geissler et al. (2022) Vaccine Solidarity

Self-contained replication package for "Public support for global
vaccine sharing in the COVID-19 pandemic: Evidence from Germany" (PLOS
ONE, 2022; doi:10.1371/journal.pone.0278337). Provides analysis data,
preparation scripts, table and figure builders, precomputed report
artifacts, and tests.
