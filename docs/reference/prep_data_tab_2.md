# Reshape wave 2 conjoint tasks into long format for Table 2

Reshape wave 2 conjoint tasks into long format for Table 2

## Usage

``` r
prep_data_tab_2(wave2_survey = wave2_survey, vignette_labels = vignette_labels)
```

## Arguments

- wave2_survey:

  Wave 2 respondent-level data.

- vignette_labels:

  Vignette attribute lookup table.

## Value

Long-format conjoint data with demeaned attributes and ratings.
