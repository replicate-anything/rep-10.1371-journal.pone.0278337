# Resolve path to a precomputed artifact file

Resolve path to a precomputed artifact file

## Usage

``` r
artifact_file(id)
```

## Arguments

- id:

  Replication id.

## Value

Character path, or `NULL` if not baked yet (run
[`build_report()`](https://replicate-anything.github.io/rep-10.1371-journal.pone.0278337/reference/build_report.md)).
