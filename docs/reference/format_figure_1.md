# Format a figure for display

Pass-through for ggplot objects produced by `make_figure_*()` (Figures
1–8).

## Usage

``` r
format_figure_1(object, ...)

format_fig_1(object, ...)

format_figure_2(object, ...)

format_fig_2(object, ...)

format_figure_3(object, ...)

format_fig_3(object, ...)

format_figure_4(object, ...)

format_fig_4(object, ...)

format_figure_5(object, ...)

format_fig_5(object, ...)

format_figure_6(object, ...)

format_fig_6(object, ...)

format_figure_7(object, ...)

format_fig_7(object, ...)

format_figure_8(object, ...)

format_fig_8(object, ...)
```

## Arguments

- object:

  ggplot object from the corresponding
  [`make_figure_1()`](https://replicate-anything.github.io/rep-10.1371-journal.pone.0278337/reference/make_figure_1.md)
  (etc.).
