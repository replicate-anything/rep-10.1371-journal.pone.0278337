# Fit marginal-effects models split by a grouping variable

Fit marginal-effects models split by a grouping variable

## Usage

``` r
fit_marginal_effects_by(data, split_var)
```

## Arguments

- data:

  Wave 4 conjoint data.

- split_var:

  Character column name to split on.

## Value

Tibble with an extra grouping column.
